Game installer creation with NSIS

NSIS (Nullsoft Scriptable Install System) is a professional open source system to create Windows installers. It is designed to be as small and flexible as possible and is therefore very suitable for internet distribution.

Being a user's first experience with your product, a stable and reliable installer is an important component of successful software. With NSIS you can create such installers that are capable of doing everything that is needed to setup your software.

Testing on a Virtual Machine

To test a game on different operating systems and previous versions, it is recommended to use a Virtual Machine (VM). Virtual machines provide a secure and isolated environment for testing software without affecting the main operating system of your machine. In this case, we will use VirtualBox as the tool to create and manage the virtual machine.

Step 1: VirtualBox Installation

  1. Download and install VirtualBox from the official Oracle website: www.virtualbox.org.
  2. Make sure to download the platform package that corresponds to your current operating system (Windows, macOS, Linux, Solaris, or OpenSolaris).
  3. Follow the installation instructions provided by the VirtualBox installation wizard.

Step 2: Creating a New Virtual Machine

  1. Open VirtualBox after successfully installing it.
  2. Click on the "New" button to create a new virtual machine.
  3. The creation wizard will guide you through the process. Ensure you provide a descriptive name for your virtual machine and select the type and version of the operating system you want to test.
  4. Set the amount of RAM you want to allocate to the virtual machine. It is recommended to allocate at least the minimum required by the operating system you will install.
  5. Select "Create a virtual hard disk now" and click "Create" to proceed with the creation of the virtual hard disk.

Step 3: Virtual Machine Configuration

  1. In the virtual hard disk creation wizard, choose the type of hard disk file you want to use. It is recommended to select "VDI (VirtualBox Disk Image)".
  2. Select the "Dynamically allocated" option for disk storage and set the maximum size you want to assign to the virtual hard disk.
  3. Click "Create" to finish creating the virtual hard disk.

Step 4: Operating System Installation

  1. Select the virtual machine you created from the VirtualBox list and click the "Start" button.
  2. You will be prompted to select an ISO file of the operating system you want to install on the virtual machine. Ensure you have a copy of the ISO file on your computer and select the corresponding location.
  3. Follow the instructions of the operating system installation wizard within the virtual machine. This may vary depending on the operating system you are installing.
  4. Complete the operating system installation process by following the on-screen instructions.
